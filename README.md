## 🔍 Troubleshooting

### The preset does not show up
Check that the preset file is in the correct folder. Reopen the game and look for the ReShade menu again.

### The screen looks black or broken
Remove the last file you copied, then start the game again. Recheck the preset path and try a different preset from the package.

### The frame rate is too low
Lower in-game graphics settings first. Then reduce sharpening, depth effects, and heavy shader effects in the preset.

### The game crashes on launch
Remove recent visual files and test the game with a clean folder. If it works, add files back one group at a time.

### The colors look too strong
Open the preset menu and reduce saturation, contrast, or bloom in small steps.
